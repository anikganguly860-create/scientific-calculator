# Ultra Advanced Scientific Calculator

A **fully-featured, ultra-fast, modern scientific calculator** built with HTML, CSS, and JavaScript. Designed to run perfectly in **mobile browsers** and on desktop. Includes live graphing, a digital clock, dark/light theme toggle, currency conversion, and full scientific functions.

---

## 🌟 Features

### Calculator Functions
- Basic arithmetic: `+`, `-`, `*`, `/`
- Scientific functions:
  - Trigonometric: `sin`, `cos`, `tan`, `sec`, `cosec`, `cot` (degrees)
  - Logarithmic: `log` (base 10), `ln` (natural)
  - Square root: `√`, factorial: `n!`
- Parentheses support for complex expressions
- **x button** for graphing expressions

### Graphing
- Plot any function with variable `x`
- Zoom and pan enabled
- Optional **open/close graph panel** for better usability

### UI & UX
- Live digital clock
- Dark and Light theme toggle
- Animated buttons for smooth interaction
- Ultra-fast and smooth performance
- Mobile-friendly layout

### Currency Conversion
- Convert between major world currencies
- Switch between **Calculator mode** and **Currency mode**
- Includes: USD, EUR, GBP, INR, JPY, CNY, AUD, CAD, CHF, SGD, NZD, MXN, BRL, ZAR, RUB, KRW
- Easy back-and-forth switching
